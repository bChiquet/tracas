You can add tags to enrich the info you collect:
<div tracas-tag="here is some data"> ... </div>

The tags can have key-value data using the following syntax:
<div tracas-tag="key1=value1;key2=value2"> ... </div>

When multiple levels in a dom tree have tags, both are picked up. However, if they share the same tag key, child data has priority.
